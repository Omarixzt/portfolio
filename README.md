Omar Khateeb | Software Engineer Portfolio
Welcome to the repository of my personal portfolio website. This project is a modern, interactive platform designed to showcase my technical skills, software projects, and professional background as a Software Engineer.

Tech Stack
The portfolio is built with performance and aesthetics in mind, utilizing modern web technologies:

Core: HTML5, CSS3, and JavaScript (ES6+).

Animations: GSAP (GreenSock) powers high-end animations and scroll-triggered effects via ScrollTrigger.

Typing Effect: Typed.js provides the dynamic typing effect in the hero section.

Icons: Lucide Icons for a clean, minimalist UI.

Deployment and Forms: Integrated with Formspree for direct email communication from the contact form.

Key Features
Dynamic Project Rendering: Projects are managed as an array of objects in JavaScript, allowing for easy updates and maintenance.

Interactive UI: Features Magnetic Buttons that react to mouse movements and custom hover effects.

Responsive Design: Fully optimized for mobile, tablet, and desktop views.

Video Previews: Project cards support auto-playing video previews on hover for desktop or intersection for mobile.

Custom Background: An interactive particle system rendered on an HTML5 Canvas.

Project Structure
index.html: The structural foundation of the site.

style.css: Contains custom styling, CSS variables, and responsive media queries.

script.js: Handles animation logic, project rendering, and interactive UI effects.

Section Overview
Hero Section: An engaging introduction featuring a typing effect and social links.

About Me: A summary of my professional journey with a creative Code Block visualization.

Skills: Highlighted technical expertise including Flutter, Node.js, and SQL.

Featured Projects: A grid of my latest work, featuring Mobile, Web, and Desktop solutions.

Contact: A clean, functional form for professional inquiries.
